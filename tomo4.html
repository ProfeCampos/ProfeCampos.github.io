🧠 El Protagonista de Hoy: El Sensor PIR

El sensor PIR (Sensor Infrarrojo Pasivo) detecta cambios en la radiación infrarroja (calor) del entorno. Es el mismo tipo de sensor que usan las luces automáticas y los sistemas de alarma para detectar la presencia de personas.

/* Misión 04: El Guardián Silencioso */
const int pinPir = 2;
const int pinLed = 13;
int estadoPir = 0; 

void setup() {
  pinMode(pinLed, OUTPUT);
  pinMode(pinPir, INPUT);
}
void loop() {
  estadoPir = digitalRead(pinPir);
  if (estadoPir == HIGH) {
    digitalWrite(pinLed, HIGH);
  } else {
    digitalWrite(pinLed, LOW);
  }
  delay(500);
}

💡 Conceptos Clave de la Misión

  • Sensor Pasivo: No emite energía, solo la recibe, lo que lo hace de muy bajo consumo.
  • Calibración: Al encenderse, el sensor necesita un tiempo para "aprender" el estado normal del ambiente antes de detectar cambios.

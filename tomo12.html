🧠 Los Protagonistas: DS18B20 y Pantalla OLED

El DS18B20 es un termómetro digital de alta precisión que se comunica por un protocolo de 1-cable. La pantalla OLED ofrece un contraste perfecto y bajo consumo porque cada píxel emite su propia luz, ideal para interfaces gráficas nítidas.

#define ONE_WIRE_BUS 2
OneWire oneWire(ONE_WIRE_BUS);
DallasTemperature sensors(&oneWire);
Adafruit_SSD1306 display(128, 64, &Wire, -1);

void setup() {
  sensors.begin();
  display.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  display.clearDisplay();
}
void loop() {
  sensors.requestTemperatures(); 
  float tempC = sensors.getTempCByIndex(0);
  
  display.clearDisplay();
  display.setTextSize(1);
  display.setTextColor(SSD1306_WHITE);
  display.setCursor(0, 0);
  display.println("Termometro Digital");
  display.setTextSize(3);
  display.setCursor(15, 25);
  display.print(tempC, 1);
  display.print(" C");
  display.display();
  delay(2000);
}

🔌 Manos a la Obra: El Circuito

Construiremos un termómetro de alta definición. Es **obligatorio** usar una resistencia de 4.7kΩ como "pull-up" entre el pin de datos del sensor y el de 5V para que la comunicación One-Wire funcione.

💡 Conceptos Clave de la Misión

  • Protocolo One-Wire: Un bus de comunicación que permite que múltiples dispositivos se comuniquen con el microcontrolador a través de un solo cable de datos.
  • Pantalla OLED: Diodo Orgánico Emisor de Luz. No necesita luz de fondo, lo que resulta en negros verdaderos y alto contraste.

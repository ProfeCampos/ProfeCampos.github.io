Tomo 7: Sintiendo el Calor - Sensor LM35 y Transistores

← Tomo Anterior Índice Tomo Siguiente →

¡Bienvenidos de vuelta, ingenieros! En la misión anterior aprendimos a controlar la posición de un motor de forma manual. Pero en la industria, la automatización real ocurre cuando el circuito toma decisiones por sí mismo basándose en lo que pasa a su alrededor.

Hoy vamos a crear un Sistema de Enfriamiento Automático. Aprenderemos a medir la temperatura ambiental real y a activar un ventilador cuando haga demasiado calor. Para lograrlo, conoceremos a dos componentes indispensables en la instrumentación.

🧠 Protagonista 1: El Sensor LM35 (El Termómetro Integrado)

El LM35 es un sensor de temperatura analógico de gran precisión. Lo genial de este pequeño componente es que viene calibrado de fábrica directamente en grados Celsius.

  • ¿Cómo funciona?: Por cada grado Celsius que aumente la temperatura, el sensor entrega 10mV (milivoltios) en su pin de señal. Por ejemplo, si estamos a 25°C, el sensor entregará 250mV.
  • Lectura en Arduino: Recibimos este voltaje mediante analogRead() en un pin analógico. Después, mediante una pequeña fórmula matemática, convertimos esa lectura digital a grados Celsius (°C) reales para poder usarla en nuestro código.

⚙️ Protagonista 2: El Transistor 2N2222 (El Interruptor Electrónico)

Un pin de Arduino solo puede entregar un máximo de 40mA de corriente. Si conectas un motor o un ventilador directo a un pin, ¡podrías quemar tu Arduino! Aquí es donde entra el transistor.

  • ¿Cómo funciona?: Piensa en él como un interruptor digital o un relevador de alta velocidad. Tiene 3 patitas: Emisor, Base y Colector. Al inyectar una corriente muy pequeña en la Base (desde un pin de Arduino), cerramos el circuito entre el Colector y el Emisor, permitiendo que pase una corriente mucho mayor externa para alimentar motores o cargas pesadas.
  • Estado de Saturación: En este proyecto usaremos el transistor en modo "saturación", lo que significa que actuará simplemente como un switch cerrado (encendido) cuando el pin del Arduino esté en HIGH.

🔌 El Circuito: Sistema de Ventilación Automática

Conectaremos el LM35 para monitorear la temperatura ambiental. Si la temperatura supera un límite establecido, el Arduino mandará una señal al transistor para encender el ventilador de DC.

Componentes Necesarios:

  • 1 Arduino UNO y Protoboard
  • 1 Sensor de temperatura LM35
  • 1 Transistor NPN (2N2222 o similar)
  • 1 Resistencia de 1kΩ (para proteger la base del transistor)
  • 1 Motor o Ventilador de DC (5V)
  • 1 Diodo 1N4007 (Diodo de protección contra contracorrientes del motor)
  • Cables Jumper

📝 Instrucciones de Conexión:

  1. Sensor LM35:
    • Pin 1 (VCC): a los 5V del Arduino.
    • Pin 2 (VOUT/Señal): al pin analógico A0.
    • Pin 3 (GND): al GND del Arduino.
  2. Circuito de Potencia (Transistor y Motor):
    • Base (Pin central del 2N2222): Se conecta a una resistencia de 1kΩ, y el otro extremo de la resistencia va al pin digital 2 del Arduino.
    • Emisor: Va directo a la línea de GND del Arduino.
    • Colector: Se conecta al cable **negativo** del motor.
    • El cable **positivo** del motor va directo a los 5V.
    • Nota de protección: Coloca el diodo 1N4007 en paralelo con el motor, con la franja gris apuntando al cable positivo. ¡Esto protege al circuito de picos de voltaje!

💡 Conceptos Clave de la Misión

  • Conversión Analógica a Digital (ADC): Arduino convierte el voltaje de entrada (0-5V) en un número entero (0-1023) mediante analogRead().
  • Factor de Escala del LM35: Como el sensor entrega 10mV por cada grado, la fórmula matemática para obtener los grados Celsius es: (voltaje_en_milivoltios / 10.0).
  • Aislamiento de Corriente: El transistor permite controlar un componente de alto consumo utilizando una señal de control de muy baja corriente desde el microcontrolador.

🧠 El Código: Control de Temperatura Inteligente

En este código leeremos de manera constante el valor analógico del sensor, haremos las operaciones matemáticas para guardarlo en una variable tipo float (con punto decimal) y usaremos un condicional if para decidir si encendemos el ventilador.

Código Arduino 

/*
 * Misión 07: Sistema de Enfriamiento Automático
 * Descripción: Mide temperatura con LM35 y activa un ventilador mediante un transistor.
 * Por: Profe Campos
 * CECyTEM 05 Guacamayas
*/

// --- ZONA DE CONFIGURACIÓN ---
const int pinSensor = A0;     // Pin analógico donde conectamos el LM35
const int pinVentilador = 2;  // Pin digital que controla la base del transistor

int lecturaRaw;               // Variable para almacenar el valor bruto (0-1023)
float voltaje;                // Variable para almacenar el voltaje calculado
float temperatura;            // Variable para almacenar la temperatura final en °C

// Umbral de temperatura para activar el ventilador
const float UMBRAL_CALOR = 30.0; 

void setup() {
  Serial.begin(9600);                // Iniciamos la comunicación serial
  pinMode(pinVentilador, OUTPUT);    // Configuramos el pin del transistor como salida
}

void loop() {
  // 1. LEER EL SENSOR
  lecturaRaw = analogRead(pinSensor);

  // 2. CONVERTIR LECTURA A VOLTAJE (en milivoltios)
  // Multiplicamos por 5000 (porque son 5V o 5000mV) y dividimos entre los 1023 pasos del ADC
  voltaje = (lecturaRaw / 1023.0) * 5000.0;

  // 3. CALCULAR TEMPERATURA
  // Sabiendo que el LM35 da 10mV por cada 1°C, dividimos el voltaje entre 10
  temperatura = voltaje / 10.0;

  // 4. MOSTRAR DATOS EN MONITOR SERIAL
  Serial.print("Lectura ADC: ");
  Serial.print(lecturaRaw);
  Serial.print(" -> Temperatura: ");
  Serial.print(temperatura);
  Serial.println(" C");

  // 5. TOMAR DECISIÓN (Control On/Off)
  if (temperatura >= UMBRAL_CALOR) {
    digitalWrite(pinVentilador, HIGH); // Activamos el transistor -> Enciende ventilador
    Serial.println("¡Alerta de calor! Ventilador ENCENDIDO.");
  } else {
    digitalWrite(pinVentilador, LOW);  // Desactivamos el transistor -> Apaga ventilador
    Serial.println("Temperatura normal. Ventilador APAGADO.");
  }

  delay(1000); // Esperamos 1 segundo entre lecturas para no saturar la pantalla
}

🚀 ¡Inténtalo Tú Mismo! (Retos)

  • Alerta Visual y Sonora: Agrega un LED rojo y un Buzzer al circuito. Haz que se activen de forma intermitente únicamente cuando el ventilador esté encendido, alertando de forma visual y audible que el sistema está caliente.
  • Histéresis (Reto Avanzado): Si la temperatura oscila exactamente en 30.0°C, el ventilador se encenderá y apagará muy rápido, lo cual puede dañarlo. Modifica las condiciones para que el ventilador se encienda a los 30.0°C pero solo se apague cuando la temperatura baje de 28.0°C.
  • Ventilador Inteligente (Velocidad Variable): En lugar de usar digitalWrite(), conecta el transistor a un pin PWM (como el pin 3) y usa analogWrite() y la función map() para que el ventilador sople despacio a los 25°C y vaya aumentando su velocidad conforme haga más calor.

🧠 Los Protagonistas: Potenciómetro y Servo

Un potenciómetro nos da una entrada analógica variable (0-1023). Un servomotor nos da una salida de movimiento angular preciso (0-180°). El reto es "mapear" o traducir el rango del potenciómetro al rango del servo para un control intuitivo.

/* Misión 07: Control Fino */
#include 
Servo miServo;
const int pinPot = A0;
const int pinServo = 9;
int valorPot;
int angulo;

void setup() {
  miServo.attach(pinServo); 
}
void loop() {
  valorPot = analogRead(pinPot);
  angulo = map(valorPot, 0, 1023, 0, 180);
  miServo.write(angulo);
  delay(15); 
}

💡 Conceptos Clave de la Misión

  • Librerías (`#include`): Bloques de código pre-hecho que nos dan funcionalidades extra, como controlar servos de manera sencilla.
  • Función `map()`: Una herramienta muy útil que re-escala un número de un rango a otro.

🧠 Los Protagonistas: DHT11, Relé y LCD I2C

El DHT11 mide temperatura y humedad con un solo pin digital. El Relé es un interruptor electromagnético que nos permite controlar aparatos de alto voltaje (como los de casa) de forma segura. La pantalla LCD con módulo I2C nos permite mostrar información usando solo 2 pines de datos.

/* Misión 09: El Panorama Completo */
#include 
#include 
#include "DHT.h"

#define DHTPIN 2
#define DHTTYPE DHT11
const int pinRele = 7;

DHT dht(DHTPIN, DHTTYPE);
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  dht.begin();
  lcd.init();
  lcd.backlight();
  pinMode(pinRele, OUTPUT);
}
void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  lcd.clear();
  lcd.print("Temp: "); lcd.print(t); lcd.print(" C");
  lcd.setCursor(0, 1);
  lcd.print("Hum:  "); lcd.print(h); lcd.print(" %");
  delay(2000);
}

🔌 Manos a la Obra: El Circuito

Crearemos una estación de monitoreo que muestra los datos en la LCD. El relé se conecta al aparato a controlar interrumpiendo uno de sus cables de alimentación y conectando los extremos a los puertos COM y NO (Normalmente Abierto).

💡 Conceptos Clave de la Misión

  • Aislamiento Eléctrico: El relé aísla completamente el circuito de control de bajo voltaje del Arduino del circuito de potencia de alto voltaje, lo que es crucial por seguridad.
  • Comunicación I2C: Un protocolo de comunicación que permite conectar múltiples dispositivos (sensores, pantallas) usando solo dos pines (SDA y SCL).

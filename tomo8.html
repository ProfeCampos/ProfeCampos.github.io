🧠 Los Protagonistas: LM35 y Transistor

El LM35 es un sensor de temperatura de precisión cuyo voltaje de salida es proporcional a los grados Celsius. El transistor actúa como un interruptor electrónico, permitiéndonos usar una señal pequeña de Arduino para controlar un dispositivo de mayor potencia, como un ventilador.

/* Misión 08: Sintiendo el Calor */
const int pinSensor = A0;
const int pinVentilador = 9;
float voltaje;
float temperaturaC;
const float umbralTemp = 28.0;

void setup() {
  pinMode(pinVentilador, OUTPUT);
}
void loop() {
  voltaje = (analogRead(pinSensor) / 1023.0) * 5.0;
  temperaturaC = voltaje * 100;
  
  if (temperaturaC > umbralTemp) {
    digitalWrite(pinVentilador, HIGH);
  } else {
    digitalWrite(pinVentilador, LOW);
  }
  delay(1000);
}

Construiremos un termostato automático. Es crucial incluir un diodo en "paralelo inverso" con el motor para proteger el transistor de picos de voltaje.

💡 Conceptos Clave de la Misión

  • Transistor como Interruptor: Usar una pequeña corriente en la 'Base' para controlar un flujo grande de corriente entre el 'Colector' y el 'Emisor'.
  • Diodo de Protección (Flyback): Absorbe la energía que un motor genera al apagarse, protegiendo los componentes del circuito.

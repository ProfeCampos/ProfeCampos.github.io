Tomo 5: El Ojo de la Noche - El Sensor de Luz LDR

← Volver al Índice Tomo Siguiente →

Nuestros proyectos ya pueden reaccionar al tacto y al movimiento. Hoy les daremos la capacidad de "ver" la luz. Usaremos una LDR (Fotoresistencia) para crear un sistema que sepa si es de día o de noche, sentando las bases para proyectos como luces nocturnas automáticas o robots que siguen la luz.

🧠 Protagonista de Hoy: La Fotoresistencia (LDR)

Una LDR (Light Dependent Resistor) es una resistencia muy especial: su valor en Ohms cambia drásticamente según la cantidad de luz que la ilumina. A más luz, menor resistencia; a mayor oscuridad, mayor resistencia. El Arduino no puede medir resistencia directamente, pero sí puede medir voltaje. Por eso, usamos un Divisor de Voltaje (la LDR y una resistencia fija de 10kΩ) para convertir ese cambio de resistencia en un cambio de voltaje que podemos leer con un pin analógico.

🧠 El Código: Midiendo la Oscuridad

Usaremos la función `analogRead()` para obtener un valor entre 0 y 1023 que representa el nivel de luz. Luego, con un `if`, decidiremos si hay suficiente oscuridad para encender un LED.


/*
 * Misión 05: El Ojo de la Noche
 * Descripción: Enciende un LED automáticamente cuando está oscuro.
 * Por: Profe Campos
 * CECyTEM 05 Guacamayas
*/

const int pinLdr = A0;      // Pin analógico para leer el divisor de voltaje.
const int pinLed = 9;       // Usamos el pin 9 que es PWM (~).

int valorLuz = 0;           // Variable para guardar la lectura del LDR.
int umbralOscuridad = 300; // Umbral. Si la lectura es MENOR a esto, está oscuro.
                            // Este valor puede necesitar ajuste.

void setup() {
  pinMode(pinLed, OUTPUT);
  Serial.begin(9600); // Para poder ver los valores del sensor.
}

void loop() {
  // Leemos el valor del pin analógico y lo guardamos.
  valorLuz = analogRead(pinLdr);
  
  // Imprimimos el valor en el Monitor Serial para poder calibrar el umbral.
  Serial.print("Valor de Luz: ");
  Serial.println(valorLuz);

  // Si el valor de luz es menor que nuestro umbral...
  if (valorLuz < umbralOscuridad) {
    digitalWrite(pinLed, HIGH); // ...encendemos la luz.
  } else { // Si no...
    digitalWrite(pinLed, LOW); // ...la apagamos.
  }
  delay(200); // Pequeña pausa.
}

🔌 Manos a la Obra: El Circuito

Construiremos una lámpara nocturna automática. El corazón del circuito es el divisor de voltaje formado por la LDR y la resistencia de 10kΩ. El punto medio es el que conectamos al pin A0.

💡 Conceptos Clave de la Misión

  • Lectura Analógica: Medición de un voltaje continuo que el Arduino convierte a un valor numérico (0-1023).
  • Divisor de Voltaje: Un circuito simple con dos resistencias en serie que nos permite obtener un voltaje de salida proporcional a la relación entre las resistencias.

🚀 ¡Inténtalo Tú Mismo! (Retos)

  • Luz de Emergencia Inversa: Haz que el LED se encienda cuando haya MUCHA luz, como una alarma que te avisa si alguien enciende la luz de tu cuarto.
  • Luz Regulable: En lugar de solo encender o apagar el LED, usa `analogWrite()` para que el brillo del LED aumente a medida que se hace más oscuro.
  • Calibración Automática: Al inicio del programa, haz que el Arduino mida la luz por 5 segundos y use ese valor promedio para establecer el `umbralOscuridad` automáticamente.

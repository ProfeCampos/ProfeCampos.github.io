🧠 El Protagonista de Hoy: La Fotoresistencia (LDR)

Una LDR (Light Dependent Resistor) es una resistencia que cambia su valor según la cantidad de luz que recibe. Para leerla, usamos un "divisor de voltaje", que nos permite convertir ese cambio de resistencia en un voltaje que Arduino sí puede medir con un pin analógico.

/* Misión 05: El Ojo de la Noche */
const int pinLdr = A0;
const int pinLed = 9;
int valorLuz = 0;
int umbralOscuridad = 300;

void setup() {
  pinMode(pinLed, OUTPUT);
}
void loop() {
  valorLuz = analogRead(pinLdr);
  if (valorLuz < umbralOscuridad) {
    digitalWrite(pinLed, HIGH);
  } else {
    digitalWrite(pinLed, LOW);
  }
  delay(200);
}

💡 Conceptos Clave de la Misión

  • Lectura Analógica: Medición de un voltaje continuo que se convierte a un valor numérico (0-1023 en Arduino).
  • Divisor de Voltaje: Un circuito simple con dos resistencias en serie que nos permite obtener un voltaje de salida proporcional a la relación entre las resistencias.

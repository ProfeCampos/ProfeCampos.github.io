🧠 La Lógica del Proyecto: El Poder del `&&` (Y)

Vamos a combinar todo lo aprendido en el primer parcial. La luz del pasillo solo debe encenderse si se cumplen DOS condiciones a la vez: que se detecte movimiento Y que esté oscuro. Esto se logra con el operador lógico `&&` (AND).

/* PROYECTO INTEGRADOR 1: Iluminación Inteligente de Pasillos */
const int pinPir = 2;
const int pinLdr = A0;
const int pinLed = 9;
int valorLuz;
boolean hayMovimiento;
int umbralOscuridad = 400; 

void setup() {
  pinMode(pinLed, OUTPUT);
  pinMode(pinPir, INPUT);
}
void loop() {
  valorLuz = analogRead(pinLdr);
  hayMovimiento = digitalRead(pinPir);
  
  if (hayMovimiento == true && valorLuz < umbralOscuridad) {
    digitalWrite(pinLed, HIGH);
    delay(5000); // Mantiene la luz encendida 5 segundos
  } else {
    digitalWrite(pinLed, LOW);
  }
  delay(300);
}

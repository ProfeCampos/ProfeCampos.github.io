🧠 La Lógica del Proyecto: Menú Interactivo

Construiremos un dispositivo que mide temperatura, humedad, presión y altitud. Usaremos un Joystick para navegar entre diferentes "pantallas" o "estados" en la pantalla OLED, mostrando la información de forma ordenada.

Adafruit_SSD1306 display(128, 64, &Wire, -1);
DHT dht(2, DHT22);
// Adafruit_BMP085 bmp; // (ejemplo para BMP180)

int pantallaActual = 0;

void setup() {
  // ... iniciar todos los sensores y la pantalla ...
}
void loop() {
  // --- Lógica para navegar con el joystick ---
  int valorY = analogRead(A1); // Leer eje Y del joystick
  if (valorY > 900) { pantallaActual = 1; }
  if (valorY < 100) { pantallaActual = 0; }
  
  display.clearDisplay();
  
  if (pantallaActual == 0) {
    // --- Mostrar Temp/Humedad ---
    float t = dht.readTemperature();
    float h = dht.readHumidity();
    display.setTextSize(2);
    display.setCursor(5, 20);
    display.print(t, 1); display.println(" C");
    display.setCursor(5, 45);
    display.print(h, 1); display.println(" %");
  } else {
    // --- Mostrar Presión/Altitud ---
    display.setTextSize(1);
    display.println("--Presion/Altitud--");
    // ... código para leer y mostrar presión y altitud ...
  }
  
  display.display();
  delay(100);
}

Integraremos múltiples sensores I2C (OLED, BMP180) conectándolos en paralelo a los pines A4 (SDA) y A5 (SCL). El joystick se leerá con pines analógicos para los ejes y un pin digital para el botón.

💡 Conceptos Clave de la Misión

  • Máquina de Estados: Un modelo de programación donde el sistema se encuentra en uno de varios "estados" (pantallas) y cambia entre ellos basado en las entradas (el joystick).
  • Interfaz de Usuario (UI): Diseñar cómo el usuario interactúa con el dispositivo, lo que es clave para que un proyecto sea útil y fácil de usar.

🌟 El Protagonista: El LED

El LED (Diodo Emisor de Luz) es el "hola mundo" de la electrónica. Es un componente semiconductor que emite luz cuando la corriente fluye a través de él. A diferencia de un foco tradicional, el LED es extremadamente eficiente y tiene polaridad: una pata larga (ánodo, +) y una pata corta (cátodo, -). ¡Si lo conectas al revés, no encenderá!

🔌 El Circuito: Manos a la Obra

Vamos a construir el cuerpo de nuestra operación. Necesitamos darle al código algo físico que controlar.

Componentes Necesarios:

  • Placa Arduino UNO: Nuestro cerebro principal.
  • Protoboard: Nuestro campo de construcción.
  • LED (cualquier color): Nuestro foquito de luz.
  • Resistencia de 220Ω: El "guardaespaldas" del LED.
  • Cables Jumper: Las "venas" que llevan la electricidad.

📝 Instrucciones de Conexión:

  1. Coloca el LED: Insértalo en la protoboard.
  2. Conecta la resistencia: En la misma fila de la pata larga (+), coloca la resistencia de 220Ω hacia una fila vacía.
  3. Lleva la energía: Conecta un cable desde el Pin 13 del Arduino a la fila libre de la resistencia.
  4. Cierra el circuito: Conecta un cable desde la pata corta (-) del LED al pin GND del Arduino.

💡 Conceptos Clave

  • Resistencia: Actúa como un "guardaespaldas". Frena el flujo excesivo de electrones para evitar que el LED se queme.
  • GND (Tierra): Es la "meta" del circuito. Sin este camino de retorno, la electricidad no puede fluir.
  • HIGH y LOW: Comandos básicos. HIGH envía 5V (encendido), LOW envía 0V (apagado).

🧠 El Código: El Cerebro de la Operación

Este es el "idioma" que usaremos para darle instrucciones a nuestra placa.

Código Arduino - Misión 01: Blink 

/*
 * Misión 01: Blink - Hacer parpadear un LED
 * Por: Profe Campos
 * CECyTEM 05 Guacamayas
*/

// El bloque setup() se ejecuta UNA SOLA VEZ al iniciar.
void setup() {
  // Configuramos el Pin 13 como SALIDA (OUTPUT)
  pinMode(13, OUTPUT); 
}

// El bloque loop() se repite infinitamente.
void loop() {
  digitalWrite(13, HIGH); // Encender LED (envía 5V)
  delay(1000);            // Esperar 1 segundo (1000 milisegundos)
  digitalWrite(13, LOW);  // Apagar LED (envía 0V)
  delay(1000);            // Esperar 1 segundo
}

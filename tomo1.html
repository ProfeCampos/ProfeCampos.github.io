¡Bienvenidos a su primera misión! Hoy daremos el primer paso en el mundo de la electrónica programable. Nuestra tarea es simple pero fundamental: aprender a controlar la electricidad para darle vida a un componente. Haremos que un LED parpadee. Este es el "Hola, Mundo" de la electrónica.

🧠 El Cerebro de la Operación: El Código

Todo lo que hacemos con Arduino necesita un plan, y ese plan es el código. Este es el "idioma" que usaremos para darle instrucciones a nuestra placa.


/*
 * Misión 01: Blink - Hacer parpadear un LED
 * Por: Profe Campos
 * CECyTEM 05 Guacamayas
*/

// El bloque setup() se ejecuta UNA SOLA VEZ cuando enciendes
// o reseteas tu Arduino. Es para configurar lo que vamos a usar.
void setup() {
  // Le decimos al Arduino: "Oye, el Pin número 13 será una SALIDA".
  // Es como abrir una compuerta para que la electricidad pueda salir.
  pinMode(13, OUTPUT);
}

// El bloque loop() es un ciclo infinito. ¡Se repite para siempre!
// Aquí ocurre toda la acción principal de nuestro programa.
void loop() {
  // 1. Encendemos el LED:
  // Enviamos una señal ALTA (HIGH) de 5V al Pin 13. ¡Luz!
  digitalWrite(13, HIGH);

  // 2. Esperamos un segundo:
  // Pausa de 1000 milisegundos (1 segundo).
  delay(1000);

  // 3. Apagamos el LED:
  // Enviamos una señal BAJA (LOW) de 0V al Pin 13. ¡Oscuridad!
  digitalWrite(13, LOW);

  // 4. Esperamos otro segundo:
  delay(1000);

  // Y... el loop() vuelve a empezar. ¡Para siempre!
}

Ahora vamos a construir el cuerpo de nuestra operación. Necesitamos darle al código algo físico que controlar. ¡Recuerda que el LED tiene una pata larga (+) y una corta (-)!

💡 Conceptos Clave de la Misión

  • Resistencia: Actúa como un "guardaespaldas" que limita la corriente para que el LED no se queme. ¡Siempre úsala!
  • GND (Tierra): Es el camino de regreso para la electricidad. Sin él, no hay flujo y nada funciona.
  • HIGH y LOW: Son las órdenes digitales básicas. HIGH es "encendido" (5V) y LOW es "apagado" (0V).

🚀 ¡Inténtalo Tú Mismo! (Retos)

  • Flash Rápido: Modifica los valores en `delay()` para que el LED parpadee mucho más rápido.
  • Latido de Corazón: Haz que el LED haga dos parpadeos rápidos seguidos de una pausa larga.
  • S.O.S.: El código morse para S.O.S. es "3 cortos, 3 largos, 3 cortos". ¿Puedes programarlo?

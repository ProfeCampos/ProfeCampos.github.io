🧠 El Cerebro de la Operación: El Código

Todo lo que hacemos con Arduino necesita un plan, y ese plan es el código. Este es el "idioma" que usaremos para darle instrucciones a nuestra placa. A continuación, analizaremos el código completo línea por línea.

/* Misión 01: Blink - Hacer parpadear un LED */
void setup() {
  pinMode(13, OUTPUT);
}
void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

💡 Conceptos Clave de la Misión

  • Resistencia: Actúa como un "guardaespaldas" que limita la corriente para que el LED no se queme.
  • GND (Tierra): Es el camino de regreso para la electricidad. Sin él, no hay flujo y nada funciona.
  • HIGH y LOW: Son las órdenes digitales básicas. HIGH es "encendido" (5V) y LOW es "apagado" (0V).

El Protagonista de Hoy: El Bucle `for`

El bucle `for` es una herramienta que hace trabajo repetitivo por nosotros de una manera eficiente. Lo usaremos para controlar varios LEDs sin tener que escribir `digitalWrite` para cada uno, creando animaciones complejas con muy poco c贸digo.

int pinesLeds[] = {2, 3, 4, 5, 6, 7, 8, 9};
int numLeds = 8;

void setup() {
  for (int i = 0; i < numLeds; i++) {
    pinMode(pinesLeds[i], OUTPUT);
  }
}
void loop() {
  // Izquierda a Derecha
  for (int i = 0; i < numLeds; i++) {
    digitalWrite(pinesLeds[i], HIGH);
    delay(80);
    digitalWrite(pinesLeds[i], LOW);
  }
  // Derecha a Izquierda
  for (int i = numLeds - 1; i >= 0; i--) {
    digitalWrite(pinesLeds[i], HIGH);
    delay(80);
    digitalWrite(pinesLeds[i], LOW);
  }
}

🧠 1. ¿Qué es el módulo LCD I2C y su conexión?

El adaptador LCD I2C permite convertir una conexión paralela de una pantalla LCD a una serie que requiere sólo de dos cables para datos[cite: 2]. Este adaptador está especialmente diseñado para conectarse con un módulo modelo 1602 (16 columnas por 2 filas)[cite: 2].

⚙️ 2. Instalación de la biblioteca LCD I2C y Funciones Útiles

Para lograr la programación del LCD con el adaptador I2C, es necesario instalar una nueva librería[cite: 2]. Esta no se encuentra disponible en el gestor normal, por lo que haremos una instalación manual por única vez:

  1. Ingresa con tu navegador web a la dirección: https://github.com/fmalpartida/New-LiquidCrystal[cite: 2].
  2. Da clic en el botón verde Code y selecciona Download ZIP[cite: 2].
  3. En el IDE de Arduino, ve al menú Programas → Incluir librería → Añadir biblioteca .ZIP[cite: 2] (busca el archivo descargado y ábrelo).
  4. Verifica la instalación buscando "New-LiquidCrystal-master" en el menú Incluir Librería[cite: 2].

🔧 La Función Constructora

Con esta librería, inicializamos el objeto definiendo los pines internos del adaptador mediante la siguiente estructura[cite: 2]:

LiquidCrystal_I2C lcd(0x27, 2, 1, 0, 4, 5, 6, 7, 3, POSITIVE);[cite: 2]

(Donde 0x27 es la dirección, y los demás números corresponden a los pines E, RW, RS, D4-D7, P3 y la polaridad para encender la luz trasera[cite: 2]).

📚 API: Funciones Útiles de la Librería

Función Descripción
lcd.init();Inicializa la interfaz a la pantalla LCD I2C y especifica la dirección y dimensiones. Debe ser llamada antes que cualquier otro comando.
lcd.backlight();Habilita (activa) el LED de retroiluminación del LCD.
lcd.noBacklight();Deshabilita (apaga) la luz de fondo del LCD.
lcd.clear();Borra la pantalla LCD y posiciona el cursor en la esquina superior izquierda.
lcd.home();Establece el cursor en la esquina superior izquierda sin borrar el texto en la pantalla.
lcd.setCursor(col, row);Posiciona el cursor del LCD en una ubicación deseada (columna, fila).
lcd.noCursor();Oculta el cursor del LCD.
lcd.cursor();Muestra el cursor del LCD si ha sido ocultado.
lcd.scrollDisplayLeft();Desplaza el contenido de la pantalla (texto y cursor) un espacio a la izquierda.
lcd.scrollDisplayRight();Desplaza el contenido de la pantalla (texto y cursor) un espacio a la derecha.

💻 3. Proyecto 1: Texto y variables en pantalla

Vamos a probar que todo funcione imprimiendo un saludo y un contador de segundos utilizando la librería que acabamos de instalar.

Código Arduino - Proyecto 1 

#include <Wire.h> 
#include <LCD.h>
#include <LiquidCrystal_I2C.h>

// Configuración de pines internos del adaptador según la librería New-LiquidCrystal
LiquidCrystal_I2C lcd(0x27, 2, 1, 0, 4, 5, 6, 7, 3, POSITIVE); 

void setup() {
  lcd.begin(16, 2); // 16 columnas por 2 lineas
  lcd.clear();      // Limpia la pantalla
}

void loop() {
  lcd.setCursor(0, 0); 
  lcd.print("Hola, han pasado"); 
  
  lcd.setCursor(0, 1); 
  lcd.print(millis() / 1000); 
  lcd.print(" seg."); 
}

🎬 4. Proyecto 2: Texto desplazable (Efecto Marquesina)

Para mostrar un mensaje largo o darle dinamismo a nuestro proyecto, usaremos la función de scroll para desplazar el texto hacia la izquierda[cite: 2].

Código Arduino - Scroll de Texto 

#include <Wire.h> 
#include <LCD.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 2, 1, 0, 4, 5, 6, 7, 3, POSITIVE); 

void setup() {
  lcd.begin(16, 2);
  lcd.clear();
}

void loop() {
  lcd.setCursor(2, 0);   
  lcd.print("Prueba scroll");  
  
  lcd.scrollDisplayLeft();  // Desplaza un carácter a la vez hacia la izquierda
  delay(1000);   
}

🎨 5. Proyecto 3: Creación de caracteres personalizados

La librería nos permite dibujar hasta 8 símbolos propios que no existen en el abecedario representados mediante matrices binarias (ceros y unos)[cite: 2].

Código Arduino - Emojis Personalizados 

#include <Wire.h> 
#include <LCD.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 2, 1, 0, 4, 5, 6, 7, 3, POSITIVE); 

// Matriz para una carita feliz
byte sonrisa[8] = {
  0b00000, 0b00000, 0b11011, 0b11011, 0b00000, 0b10001, 0b01110, 0b00000
};

// Matriz para un Alien
byte Alien[] = {
  B11111, B10101, B11111, B11111, B01110, B01010, B11011, B00000
};

void setup() {
  lcd.begin(16, 2);
  lcd.clear();
  
  // Guardamos nuestros símbolos en la memoria (Espacios 0 y 1)
  lcd.createChar(0, sonrisa);
  lcd.createChar(1, Alien);
  
  lcd.home(); // Posiciona el cursor en 0,0
  lcd.print("Robot Alien ");
  lcd.write(byte(1)); // Dibuja el Alien
}

void loop() {
  lcd.setCursor(0, 1);
  lcd.print("Feliz ");
  lcd.print(char(0)); // Dibuja la carita feliz (forma alternativa)
  delay(1000);
}

🛠️ 6. Solución de Problemas Comunes

¿Cargaste el código y no se ve nada?

¡No entres en pánico, ingeniero! Verifica estos tres puntos clave:

  • El Ajuste del Contraste: El adaptador tiene un preset (potenciómetro) para la regulación del nivel de contraste de la pantalla[cite: 2]. Toma un desarmador pequeño y gíralo suavemente hasta que las letras aparezcan.
  • Cuadros negros sólidos: Esto significa que la pantalla tiene energía, pero no está recibiendo datos o falló la inicialización. Verifica que el cable de SDA vaya correctamente a su pin y el de SCL al suyo, ¡suelen confundirse mucho!
  • Dirección I2C Incorrecta: Nuestro adaptador tiene por defecto la dirección 0x27[cite: 2]. Si tu módulo es distinto, podría usar la dirección 0x3F. Prueba cambiando el 0x27 en la línea de configuración del LiquidCrystal_I2C.

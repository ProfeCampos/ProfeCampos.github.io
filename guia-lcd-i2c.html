Guía Avanzada: Interfaz I2C para Pantalla LCD 16x2

¡Hola, ingenieros del CECyTEM! Esta es una guía de profundización completa para conectar y dominar una pantalla LCD 16x2 usando el protocolo de comunicación I2C con Arduino. Aprenderemos a dejar atrás los circuitos llenos de cables para crear proyectos limpios y profesionales.

A lo largo de esta guía, no solo mostraremos mensajes básicos; crearemos animaciones de texto desplazable, mostraremos variables de sensores en tiempo real y, para cerrar con broche de oro, ¡aprenderemos a dibujar nuestros propios caracteres personalizados (emojis) en la pantalla!

🧠 1. ¿Qué es el módulo LCD I2C y su conexión?

El estándar I2C (Inter-Integrated Circuit) es un bus de comunicación que permite a tu Arduino hablar con la pantalla usando solo dos cables de datos. Si miras detrás de tu LCD, verás una pequeña placa negra soldada: ese es el expansor I2C (usualmente el chip PCF8574).

🔌 Configuración de Pines (Solo 4 cables)

  • GND: Al pin GND de tu Arduino.
  • VCC: A los 5V de tu Arduino.
  • SDA (Datos): Al pin A4 en el Arduino UNO.
  • SCL (Reloj): Al pin A5 en el Arduino UNO.

⚙️ 2. Instalación de la biblioteca LCD I2C

Para que el Arduino entienda este protocolo de comunicación sin tener que escribir miles de líneas de código complejo, necesitamos usar una biblioteca externa. Ve a tu Arduino IDE, abre el Gestor de Librerías (Programa > Incluir Librería > Administrar Bibliotecas) y busca LiquidCrystal I2C por Frank de Brabander. ¡Instálala!

💻 3. Proyecto 1: Texto y variables en pantalla

Vamos a probar que todo funcione imprimiendo un saludo y un contador de segundos. Observa cómo usamos lcd.setCursor(columna, fila) para ubicar el texto. Recuerda que empezamos a contar desde el 0.

Código Arduino - Texto y Variables 

#include <Wire.h> 
#include <LiquidCrystal_I2C.h>

// Dirección I2C típica: 0x27 o 0x3F, pantalla de 16 columnas y 2 filas
LiquidCrystal_I2C lcd(0x27, 16, 2); 

void setup() {
  lcd.init();       // Inicia la comunicación con la pantalla
  lcd.backlight();  // Enciende la luz de fondo azul
  
  lcd.setCursor(0, 0); // Columna 0, Fila 0 (Arriba a la izquierda)
  lcd.print("Hola CECyTEM!");
}

void loop() {
  lcd.setCursor(0, 1); // Columna 0, Fila 1 (Abajo a la izquierda)
  lcd.print("Segundos: ");
  // millis() devuelve los milisegundos desde que el Arduino encendió
  lcd.print(millis() / 1000); 
}

🎬 4. Proyecto 2: Texto desplazable (Efecto Marquesina)

¿Qué pasa si queremos mostrar un mensaje de más de 16 letras? ¡Lo hacemos rodar! Usaremos la función lcd.scrollDisplayLeft() dentro de un bucle for para crear un letrero dinámico.

Código Arduino - Scroll de Texto 

#include <Wire.h> 
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2); 

void setup() {
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("Alerta: Fuera de rango de control!");
  delay(1000); // Esperamos un segundo antes de moverlo
}

void loop() {
  // Desplazamos el texto hacia la izquierda 13 posiciones
  for (int i = 0; i < 13; i++) {
    lcd.scrollDisplayLeft();
    delay(300); // Velocidad del movimiento
  }
  
  delay(1000);
  
  // Regresamos el texto a su posición original
  for (int i = 0; i < 13; i++) {
    lcd.scrollDisplayRight();
    delay(300);
  }
}

🎨 5. Proyecto 3: Creación de caracteres personalizados

La pantalla LCD nos permite crear hasta 8 símbolos propios que no existen en el abecedario. Cada símbolo se dibuja encendiendo y apagando píxeles en una cuadrícula de 5x8. Los representamos usando números binarios (donde 1 es encendido y 0 es apagado).

Código Arduino - Emojis Personalizados 

#include <Wire.h> 
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2); 

// Matriz binaria para un icono de un corazón
byte corazon[8] = {
  B00000,
  B01010,
  B11111,
  B11111,
  B01110,
  B00100,
  B00000,
  B00000
};

// Matriz binaria para una carita feliz
byte carita[8] = {
  B00000,
  B01010,
  B01010,
  B00000,
  B10001,
  B01110,
  B00000,
  B00000
};

void setup() {
  lcd.init();
  lcd.backlight();
  
  // Guardamos nuestros símbolos en la memoria de la pantalla (Espacios 0 y 1)
  lcd.createChar(0, corazon);
  lcd.createChar(1, carita);
  
  lcd.setCursor(0, 0);
  lcd.print("Robot Feliz ");
  lcd.write(0); // Dibujamos el corazón
  lcd.write(1); // Dibujamos la carita
}

void loop() {
}

🛠️ 6. Solución de Problemas Comunes

¿Cargaste el código y no se ve nada?

¡No entres en pánico! Este es el problema más común de todo instrumentista:

  • El Ajuste del Contraste: Detrás de tu módulo I2C hay un pequeño componente azul con un tornillo dorado en el centro (es un potenciómetro). Toma un desarmador de cruz pequeño y gíralo suavemente mientras el circuito está encendido hasta que las letras aparezcan.
  • La pantalla solo muestra cuadros negros sólidos: Esto significa que la pantalla tiene energía, pero no está recibiendo datos. Verifica que el cable de SDA vaya a A4 y el de SCL a A5, ¡suelen confundirse mucho!
  • Dirección I2C Incorrecta: Si los cables y el contraste están bien, puede que tu módulo no responda al "nombre" 0x27. Intenta cambiar en el código esa línea a LiquidCrystal_I2C lcd(0x3F, 16, 2);.

Guía de Dispositivo: Pantalla LCD 16x2 con Módulo I2C

¡Hola, creadores! En esta guía profunda aprenderemos a utilizar una Pantalla LCD (Display de Cristal Líquido) de 16 columnas por 2 filas. Pero no lo haremos de la forma tradicional, sino utilizando un módulo de comunicación llamado I2C, el cual nos ahorrará una enorme cantidad de cables.

🧠 ¿Qué es el protocolo I2C?

El estándar I2C (Inter-Integrated Circuit) es un bus de comunicación que permite a múltiples dispositivos "hablar" con el Arduino usando solo dos cables de datos. Un LCD normal requiere hasta 6 cables de datos al Arduino; con el módulo I2C soldado en la parte trasera, lo reducimos a solo 4 cables en total (incluyendo energía).

  • GND: Al negativo de la protoboard.
  • VCC: A los 5V de Arduino.
  • SDA (Serial Data): Por aquí viajan los datos. En el Arduino UNO va al pin A4.
  • SCL (Serial Clock): El reloj que sincroniza la comunicación. En el Arduino UNO va al pin A5.

🧠 El Código y la Librería

Para que el Arduino entienda este protocolo, necesitamos instalar una librería especial llamada LiquidCrystal_I2C. Una vez instalada, el código es muy sencillo.

Código Arduino 

#include <Wire.h> 
#include <LiquidCrystal_I2C.h>

// Crear el objeto lcd. 
// Parámetros: (Dirección I2C, número de columnas, número de filas)
// La dirección suele ser 0x27 o 0x3F.
LiquidCrystal_I2C lcd(0x27, 16, 2); 

void setup() {
  lcd.init();       // Inicializar el LCD
  lcd.backlight();  // Encender la luz de fondo
  
  lcd.setCursor(0, 0); // Posicionarse en la columna 0, fila 0
  lcd.print("Hola CECyTEM!");
  
  lcd.setCursor(0, 1); // Posicionarse en la columna 0, fila 1
  lcd.print("Ingenieros :)");
}

void loop() {
  // En este ejemplo el texto es fijo, no necesitamos nada en el loop
}

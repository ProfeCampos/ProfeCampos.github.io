🧠 El Protagonista de Hoy: El Sensor HC-SR04

Este sensor funciona como un sonar: emite un pulso de sonido ultrasónico y mide el tiempo que tarda en regresar el eco. Conociendo la velocidad del sonido, podemos calcular la distancia a un objeto con gran precisión.

/* Misión 11: Viendo con Sonido */
const int pinTrig = 9;
const int pinEcho = 10;
long duracion;
int distancia;

void setup() {
  pinMode(pinTrig, OUTPUT);
  pinMode(pinEcho, INPUT);
  Serial.begin(9600);
}
void loop() {
  digitalWrite(pinTrig, LOW);
  delayMicroseconds(2);
  digitalWrite(pinTrig, HIGH);
  delayMicroseconds(10);
  digitalWrite(pinTrig, LOW);
  
  duracion = pulseIn(pinEcho, HIGH);
  distancia = duracion * 0.0343 / 2;
  
  Serial.print("Distancia: ");
  Serial.print(distancia);
  Serial.println(" cm");
  delay(500);
}

🔌 Manos a la Obra: El Circuito

Construiremos una cinta métrica digital. El pin `Trig` envía el sonido y el pin `Echo` lo recibe. Es importante conectarlos a los pines correctos de Arduino.

💡 Conceptos Clave de la Misión

  • Ecolocalización: El principio de usar ondas (de sonido, en este caso) y sus ecos para localizar objetos en el espacio.
  • Función `pulseIn()`: Mide la duración de un pulso (cuánto tiempo un pin se mantiene en estado `HIGH` o `LOW`), ideal para este sensor.
